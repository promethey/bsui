## Installation

Requirements:

- React 18+
- Bootstrap 5.1.3

```bash
npm install bsui
```

Import Bootstrap styles in your project:

```bash
npm install bootstrap
```

```js
import "bootstrap/dist/css/bootstrap.min.css";
```

## Usage

```jsx
import { Button } from "bsui";

export function Example() {
  return (
    <Button tone="primary" size="lg">
      Primary
    </Button>
  );
}
```
