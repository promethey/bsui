# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in BSUI, please report it privately.

Please do not open a public GitHub issue for security-related problems.

## How to Report

Send a report to:

sedelkovegor@gmail.com

Include:

- Description of the vulnerability
- Steps to reproduce the issue
- Affected versions
- Possible impact
- Suggested fix (if available)

## Response Process

I will review the report and respond as soon as possible.

After confirming the issue, a fix will be prepared and a new release may be published.
